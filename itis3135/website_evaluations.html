Website Evaluations

Amazon

Amazon is a popular website that allows users to order products and have them delivered to a desired address. The website is well designed with clearly defined navigaiton tools.

  • Contrast
  • Repetition
  • Alignment
  • Proximity

This website uses a clear colorscheme and the website makes it clear what the more important aspects of the site are. The website menu remains consistant throughout all windows of the site. Products are perfectly aligned in a grid. The website makes good use of whitespace to space out elements.

Arngren

Arngren is an online store that allows users to order an assortment of random products. The page is a clustered mess and hard to navigate.

  • Contrast
  • Repetition
  • Alignment
  • Proximity

The website doesnt make good use of contrast. Everything on the page is very colorful and colors of the products clash against one another. There is no repetition on this page, links are several different colors. The only use of repetition is in the red links for the price. Other links are randomly yellow or blue. There is no alignment of the page. Product images are scattered throughout the page with absolutely no grid alignment. This page also fails to follow proximity, images sometimes are cutoff by other content of the page.
